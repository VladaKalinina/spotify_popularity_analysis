# Популярность треков Spotify

Учебный проект по анализу данных на Python.

**Команда:** Калинина Влада, Худякова Светлана

## О проекте

Мы проверяем, какие свойства музыки связаны с популярностью треков на Spotify.

Основные вопросы:

- отличаются ли по популярности треки с пометкой `explicit`;
- связана ли танцевальность с популярностью;
- популярнее ли треки в мажоре;
- можно ли оценить популярность по свойствам музыки;
- помогают ли данные Last.fm улучшить результат.

Анализ показывает связи между показателями, но не доказывает, что одно свойство напрямую повышает популярность трека.

## Данные

Основной источник — [Spotify Tracks Dataset на Kaggle](https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset). В нём около 114 тысяч треков с жанрами, популярностью и свойствами музыки.

Скачанный файл нужно поместить сюда:

```text
data/raw/dataset.csv
```

Дополнительные данные о слушателях, прослушиваниях и тегах исполнителей загружаются из Last.fm.

## Запуск

1. Создайте и включите виртуальное окружение:

```bash
python -m venv .venv
source .venv/bin/activate
```

Для Windows PowerShell:

```powershell
.venv\Scripts\Activate.ps1
```

2. Установите библиотеки:

```bash
pip install -r requirements.txt
```

3. Скачайте `dataset.csv` с Kaggle и поместите его в `data/raw/`.

4. Для загрузки данных Last.fm скопируйте `.env.example` в `.env` и добавьте свой ключ `LASTFM_API_KEY`.

5. Запустите загрузку данных Last.fm:

```bash
python src/enrich_lastfm.py
```

6. Запустите Jupyter:

```bash
jupyter lab
```

Откройте `notebooks/spotify_popularity_analysis.ipynb` и выполните все ячейки.

## Основные выводы

- Танцевальность слабо связана с популярностью.
- Треки в мажоре не оказались заметно популярнее треков в миноре.
- По одним свойствам музыки сложно точно оценить популярность.
- На популярность также влияют известность исполнителя, реклама, плейлисты, дата выхода и музыкальные тренды.

Точные значения, таблицы и графики находятся в ноутбуке.
